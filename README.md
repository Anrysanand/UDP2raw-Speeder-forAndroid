此APP需要root才能运作！

可以在Android上创建UDPspeeder以及UDP2raw的服务，把手机当作一台区域网中转服务器，可以自定command，用法和Linux没区别

如果是像我一样，用v2ray连接本地的udpspeeder进行转发的话，切勿开启全局模式代理，不然APP的通信会被覆盖，会形成本地死循环而无法连接，我无法解决这个问题，请开启手动分流，只代理需要代理的APPs

其他配置详细请去看原主
